# 排行榜系统

## 脚本路径

```text
mdtserver/config/scripts/wayzer/user/leaderboard.kts
```

## 入口

- `/rank`
- `/leaderboard`
- `/排行榜`
- `/排行`
- `/help` → 玩家指令 → **排行榜**

## 当前排行榜

排行榜读取已落盘的 MDT 数据库统计，最多显示前 10 名：

- MDC排行：当前持有 MDC。
- 累计MDC排行：历史累计获得 MDC。
- 发帖数排行：历史累计发帖数。
- 被赞排行。
- 被踩排行。
- 送出赞排行。
- 送出踩排行。
- 被认可排行。
- 认可他人排行。

## 显示规则

- 在线玩家优先显示当前名字。
- 离线玩家优先显示数据库里记录的最近名字。
- 显示名前会清理历史记录里残留的前置称号文本，例如 `[游客]`、`[成员]`，避免排行榜把所有人显示成游客头衔。
- 如果没有最近名字，则显示 UID 的前若干字符。

## 维护备注

- 排行榜只读数据，不改变玩家状态。
- 数据库查询放入 `Dispatchers.IO`。
- 后续如添加新统计项，优先在 `MdtStorage` 增加专用 top 查询接口，避免脚本层全量读取后排序。
