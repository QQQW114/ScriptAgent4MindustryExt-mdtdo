# 私聊系统

## 脚本路径

```text
mdtserver/config/scripts/wayzer/user/privateMessage.kts
```

## 入口

- `/msg <玩家3位ID/#游戏ID/名字> <内容>`：给在线玩家发送私聊。
- `/m`、`/tell`、`/w`、`/私聊`：`/msg` 的别名。
- `/r <内容>`：回复最近一次私聊对象。
- `/reply`、`/回复`：`/r` 的别名。
- `/help` → 玩家指令 → **私聊**：快速查看入口。

## 当前规则

- 1级及以上玩家可发送私聊；0级/未绑定玩家不可发送，降低机器人骚扰风险。
- 被禁言玩家不可发送私聊，避免绕过禁言。
- 不能给自己发私聊。
- 单条私聊最多保留 240 字。
- 私聊带简单限速：默认 10 秒最多 5 条，超过会提示稍后再试。
- 成功发送后双方都会看到 `[私聊]` 格式消息，接收方会提示可用 `/r <内容>` 快速回复。

## 存储

当前私聊不落盘，仅保存在线期间最近回复对象和临时限速计数；玩家离线后会清理相关状态。

## 后续可扩展

- `/ignore <玩家>`：屏蔽某人的私聊。
- `/msgtoggle`：关闭/开启接收私聊。
- 管理端审计：如确实需要，可只记录元数据，不建议默认记录私聊正文。
