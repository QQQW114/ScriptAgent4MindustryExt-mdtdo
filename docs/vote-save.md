# 投票存档

对应脚本：

- `mdtserver/config/scripts/wayzer/cmds/voteSave.kts`
- `mdtserver/config/scripts/wayzer/map/autoSave.kts`

## 功能

玩家可通过投票为当前正在进行的游戏创建存档：

- `/vote save`：自动选择一个投票存档槽位。
- `/vote save <存档ID>`：投票保存到指定投票存档槽。
- `/vote save [存档ID] [备注]`：带备注创建存档。

投票通过后，会把当前游戏写入投票存档槽。可用 `/slots` 查看自动存档与投票存档，也可用 `/vote rollback <存档ID>` 投票回档。

## 槽位约定

- 自动存档：`100-105`
- 投票存档：`106-110`
- 系统临时回档槽：`111`

投票存档只允许写入 `106-110`，避免覆盖自动存档和系统临时槽。

## 边界

- 只有游戏进行中才能创建投票存档。
- 不指定槽位时，优先使用空槽/损坏槽；如果都已存在，则覆盖最旧的投票存档。
- 默认成功创建后有 5 分钟冷却，避免频繁写盘。
- 单人在线时仍遵循现有投票系统的单人快速投票规则。
